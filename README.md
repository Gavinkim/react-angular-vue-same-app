# Friend list App.

The same App built with React, Angular, Vue.

Online Demo Angular version ->  [angular demo]


## Use Vue App  (v 2.x):

cd vue/app-list/

npm start

visit  --> localhost:8080

## Use React App (v 16.x):

cd react/app-list/

npm start

visit  --> localhost:8080


## Use Angular App (v 5.x):

cd angular/app-list/

npm start 

visit  --> localhost:8080

build:  ng build --base-href=./

----------------
anglular-cli.json file
"defaults": {
    "serve": {
      "port": 8080
    }
  }
